# Drone-Defense-Hackathon

https://www.agorize.com/fr/challenges/drone-defense-hackathon

Je travaille sur le **Challenge 3 - Détecter pour comprendre:** 
Comment exploiter les drones pour détecter, analyser et transmettre des informations critiques en temps réel lors d’incendies, d’inondations ou d’opérations de défense ?

Et plus précisément sur les 2 défis suivants: 
- **Analyse de l’environnement:** Repérer et suivre la propagation d’un gaz dans une zone.
- **Suivi de traces thermiques:** Détecter et cartographier des signatures de chaleur dans un environnement.
